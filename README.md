# LadsJam: Telegram-бот интеграции со Spotify 🎵

Асинхронный Telegram-бот, который проверяет, что сейчас слушают участники группы в Spotify, и генерирует единый визуальный коллаж из обложек треков по команде `nowplaying`.

## 🚀 Особенности

* **Асинхронность:** Бот построен на `aiogram 3.x`, `FastAPI` и `httpx` (для запросов к Spotify API), что позволяет выдерживать высокие нагрузки, не блокируя Event Loop.
* **Генерация изображений "на лету":** С помощью `Pillow` бот скачивает обложки альбомов и собирает их в красивый коллаж с динамической сеткой (адаптируется под количество слушателей). Рендеринг вынесен в `asyncio.to_thread`.
* **Автоматический трекинг участников:** Бот пассивно "слушает" сообщения в группе (сохраняет связку `telegram_id` и `group_id` в SQLite) — без необходимости ручного добавления.
* **Smart OAuth & Webhooks:** Обработка коллбеков авторизации Spotify и прием вебхуков Telegram реализовано через единое FastAPI-приложение. Токены обновляются (Refresh Token) автоматически перед каждым запросом к Spotify.

## 🏗 Архитектура приложения

Проект разбит на логические модули:

1. **`main.py`** — Точка входа (FastAPI). Запускает приложение (Uvicorn), поднимает базу данных, регистрирует Webhook для Telegram и эндпойнт `/callback` для OAuth от Spotify.
2. **`config.py`** — Настройки приложения, читает переменные окружения через `.dotenv`.
3. **`database.py`** — Асинхронные CRUD-операции с базой данных (SQLite3 через `aiosqlite`). 
    * Таблица `users`: Пользователи (Telegram ID) и их Spotify Refresh Tokens.
    * Таблица `group_members`: Связь пользователя (Telegram ID) с чатами (Group ID).
4. **`spotify_api.py`** — Инкапсулирует логику Spotify API. Авторизация, обновление токена (refresh), получение текущего трека (currently-playing) и профиля юзера.
5. **`image_builder.py`** — Графический движок на `Pillow`. Расчет размеров и колонок, наложение градиентов от черного к прозрачному для читаемости текста, написание текста (трек, артист, ник) и отдачи JPEG в памяти.
6. **`bot_handlers.py`** — Роуты `aiogram`:
    * `/start` (ЛС) — Выдача ссылки для авторизации Spotify (через OAuth).
    * `nowplaying` (Группа) — Триггер бота. Сбор участников группы, опрос Spotify API, генерация картинки, отправка фото в чат.
    * Обработчик всех сообщений (Группа) — Фоновое добавление пользователя в таблицу `group_members`, если он что-то написал в чате.

## ⚙️ Установка и запуск (Local Environment)

### Требования
* Python 3.11+
* Зарегистрированное приложение в [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
* Токен Telegram-бота (от [@BotFather](https://t.me/botfather))

### 1. Подготовка окружения
Клонируйте репозиторий и установите зависимости:
```bash
pip install -r requirements.txt
```

### 2. Переменные окружения (.env)
Создайте файл `.env` в корне проекта (рядом с `main.py`):
```env
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrSTUvwxYZ
SPOTIFY_CLIENT_ID=ваш_client_id_от_spotify
SPOTIFY_CLIENT_SECRET=ваш_client_secret_от_spotify

# URL, на который Spotify перенаправит пользователя после авторизации
# В Spotify Dashboard этот URL должен быть добавлен в список Redirect URIs
SPOTIFY_REDIRECT_URI=https://<ваш-домен-или-ngrok>/callback

# Базовый URL для Telegram Webhook
BASE_URL=https://<ваш-домен-или-ngrok>

# Путь к файлу базы данных SQLite
DB_PATH=spotifly.db
```

> **Важно:** Для локального тестирования Webhooks (если у вас нет белого IP и сертификата) потребуется использовать туннелирование, например, **ngrok** или **localtunnel**, чтобы прокинуть локальный порт (8000) наружу.

### 3. Запуск сервера

Запустите проект с помощью команды:
```bash
python main.py
```
*(Под капотом запускается Uvicorn на `0.0.0.0:8000`)*

## 🎮 Как пользоваться ботом

1. **Регистрация Spotify:** Отправьте боту в личные сообщения команду `/start`. Перейдите по ссылке, которую он пришлет, авторизуйтесь в Spotify (бот запомнит ваш токен).
2. **Добавление в группу:** Добавьте бота в любой групповой чат.
3. **Активация слежения:** Обязательно напишите хотя бы одно любое сообщение в эту группу (чтобы бот добавил вас в список участников *именно этой группы*). 
4. **Магия:** Включите музыку на любом из ваших девайсов и отправьте в группу команду `nowplaying`.
5. Бот моментально соберет данные всех активных в Spotify юзеров чата и скинет стильный коллаж! 🎵
